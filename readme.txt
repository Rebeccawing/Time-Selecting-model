Design
1. 内容说明
 【SelectTimeclass】
       把择时系统封装为一个类，并直接在code中把四个指数的择时结果运行并存在result的4张csv中
----------------------------------------------------------------------------
 【result】：所有的择时系统评分指标结果表的存放文件夹
----------------------------------------------------------------------------
 【indices_ohlc】: 从wind上下载的上证50，沪深300，沪深500，沪深800指数的开盘、收盘、最高、最低价


2. 代码的基本思想
 首先，数据处理：将原始数据从xlsx中读出来后将日期换成year作为level=0的索引，将其从0到2800的递增数字作为Level=1的索引，即：每个数据的索引都是Multi-index
 其次，按照策略求出ud以及ud_sum，此处的函数变量为n1
       将ud_sum = -n2的位置存到【买入启动数组】，将ud_sum = n2的位置存到【卖出启动数组】
 第三，按照研报逻辑将买入启动后的满足累加为n3的存入【买入信号开始数组】及【卖出信号开始数组】
       并算出每一个信号开始时间所对应的止损点位存入到【stop_loss】数组中
 第四，对每一个买入开始信号，求其止损点位和最近的卖出开始信号的最小值，作为该买入信号的结束值并存到【买入信号结束数组】
       对每一个卖出开始信号，求其止损点位和最近的买入开始信号的最小值，作为该卖出信号的结束值并存到【卖出信号结束数组】
 第五， 模型评价：模型评价中，算出了累计收益率，年化收益率，总交易次数，获胜次数，失败次数，胜率，单次平均收益率，单次获胜收益率，单次失败收益率，赔率，最大回撤，连续获胜次数，连续失败次数

3. 注意事项：
   在程序运行过程中，出现了一些出现0值以及收益率全为负或全为胜的情况，程序在基础上做了一些避免异常值的改进，此处不详述
